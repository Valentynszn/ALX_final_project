Key Features

Feature 1: Intuitive Controls

To move around the maze you can use these buttons W,A,D,S.

Use the W button to for move forward. ⬆

Use the S button to move bacwards. ⬇

Use the A button to turn to the left. ⬅

Use the D button to turn to the right. ➡

Other features will be added as we update the game

OR

Use the ARROW UP button to for move forward. ⬆

Use the ARROW DOWN button to move bacwards. ⬇

Use the ARROW LEFT button to turn to the left. ⬅

Use the ARROW RIGHT button to turn to the right. ➡

Other features will be added as we update the game

Feature 2: Realistic Graphics

Experience stunning and realistic graphics as you navigate through the maze.

Feature 3: Stunning Game play

Experience stunning and realistic gameplay that challenges your navigation skills.

About This Project

This project is a portfolio project for holberton school, and was inspired by our passion for game development. We chose C because it was our strong point and it is easier to use. Our team collaborated via GitHub and communicated through WhatsApp to ensure smooth progress and overcome challenges such as SDL initialization issues and raycasting fish eye effect.
